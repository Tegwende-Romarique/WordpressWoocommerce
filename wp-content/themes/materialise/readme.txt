ABOUT THEME
Materialise is a Responsive and SEO ready Blog theme based on Bootstrap. Highly customisable. Theme provides customisation to almost all elements. Theme option give option to customise theme layout. Materialise support theme color option. Materialise allow you to select archive and post page layout.
